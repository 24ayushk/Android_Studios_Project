# Android Studios Project

A collection of Android apps and experiments built using Android Studio, demonstrating core development concepts like UI design, activity lifecycle, and data handling. This repository is ideal for learning and showcasing practical Android development using Java/Kotlin and standard Android SDK tools.

## 🚀 Features

* **UI Design:** Implementation of modern and responsive Android layouts.
* **Activity Lifecycle:** Demonstrations of proper activity and fragment lifecycle management.
* **Data Handling:** Examples of managing app data effectively.
* **Practical Development:** Real-world usage of standard Android SDK tools.

## 🛠️ Technologies Used

* **IDE:** Android Studio
* **Languages:** Java, Kotlin
* **Build Tool:** Gradle (Kotlin DSL `build.gradle.kts`)
* **SDK:** Standard Android SDK

## 📂 Project Structure

* `app/`: Contains the main application source code, resources (XML layouts, drawables), and manifests.
* `gradle/`: Gradle wrapper configurations.
* `build.gradle.kts` / `settings.gradle.kts`: Gradle build configurations for managing dependencies and project settings.
* `get-pip.py`: Included Python utility script.

## ⚙️ Getting Started

To explore and run the projects locally, follow these steps:

### Prerequisites
* [Android Studio](https://developer.android.com/studio) installed on your machine.
* Standard Android SDK configured.
* An Android Virtual Device (AVD) or a physical Android device with USB debugging enabled.
